# Dotfiles

## Linux Install

```
sudo apt install git -y 
git clone https://github.com/jason9075/dotfiles.git ~/dotfiles
~/dotfiles/setup.sh
```

## Misc

If you want to change default bash to zsh, use:
```
chsh -s $(which zsh)
```

If you need to set user password, use:
```
sudo passwd <username>
```
